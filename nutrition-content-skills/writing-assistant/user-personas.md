# 用户画像（User Personas）

> 18-30 岁青年女性 · 三类核心受众 · 四层需求模型
> writing-assistant-skill 在每次开稿前会读这份文件，匹配出"本期写给谁"。

---

## 受众总览

- **年龄**：18-30 岁（核心 22-28 岁）
- **性别**：女性 ≥ 80%
- **地域**：一二线城市为主，三四线占 25%
- **学历**：大学本科及以上 70%
- **共同标签**：作息不规律、外卖依赖、有减肥经历、关心健康但缺时间、信息过载且对"专家"半信半疑

---

## 四层需求模型（每条稿要踩中至少一层）

| 层 | 需求 | 表现 | 我能给什么 |
|---|---|---|---|
| L1 生理 | 吃饱、不饿、不胖、不累 | 「想减但又饿」 | 具体的克数、菜谱、替换方案 |
| L2 安全 | 不踩雷、不伤身、不被骗 | 「这个能吃吗 / 这个真的有效吗」 | 辟谣、风险提示、可证伪的依据 |
| L3 认同 | 我是有审美/有自律/有知识的人 | 「我朋友圈不能太 low」 | 视觉精致 + 知识感 + 有梗 |
| L4 自我实现 | 把生活掌控感拿回来 | 「我想成为更好的自己」 | 长期主义、温和、不焦虑 |

## 画像 1：精致打工女 Lily（核心受众，占 50%）

### 基本盘
- **年龄**：25-30 岁
- **职业**：互联网 / 金融 / 咨询 / 设计 / 媒体 一线城市白领
- **收入**：月入 1.5-3.5 万
- **婚姻状态**：单身 / 同居 / 新婚无娃
- **居住**：合租或独租，自己很少做饭
- **平台**：小红书重度用户，抖音通勤刷，公众号收藏不看

### 一天
- 7:30 闹钟响 5 次才起，没时间吃早饭，瑞幸打卡
- 12:00 工位外卖，沙拉 / 麻辣烫 / 螺蛳粉轮换
- 15:00 下午茶团购的奶茶
- 20:00 加完班点轻食或日料，配剧
- 24:00 躺床刷小红书，"明天一定要早睡"
- 周末报复性赖床 + 报复性聚餐 + 周日晚 emo

### 痛点（按强度排序）
1. **想瘦但停不下来吃**：节食 → 暴食 → 自责的循环
2. **皮肤状态差**：熬夜 + 高糖 + 缺水 → 痘、黄、暗沉
3. **疲惫感长期在线**：怀疑自己缺铁/缺 B/缺 D，但又不知道到底缺什么
4. **经期问题**：周期不准、痛经、经期前疯狂想吃甜
5. **被减肥信息淹没**：生酮、轻断、减脂餐、博主带货，分不清真假
6. **"健康"需要花太多时间**：备餐、健身房、营养品全要钱要时间

### 我的稿子要给她的
- 通勤路上能看完的"今天就能用"的小招（≤ 60 秒口播 / ≤ 800 字小红书）
- "外卖也能不踩雷"系列（瑞幸/星巴克/麦麦/便利店 怎么点）
- "经期 / 早八 / 加班 / 出差" 等具体场景的吃法
- 一句"我懂你"的共情开场，再给方法

### 让她想点开的标题模板
- 「便利店通勤早餐 Top5：医生眼里及格的就这几个」
- 「奶茶不戒，但少喝这一类，皮肤先谢谢你」
- 「经期前为什么疯狂想吃甜？营养师的 3 个止糖法」
- 「打工人版热量缺口：不饿肚子也能瘦的偷懒公式」

### 让她想转发的钩子
- "替你妈/室友/姐妹收藏" 这种利他动机
- "原来不是我意志力差" 的解放感
- 截图友好的金句卡

### 不要对她做的事
- 不要让她"早 6 点起来准备健康早餐"（不可能）
- 不要"放弃所有奶茶 / 外卖 / 夜宵"（极端建议劝退）
- 不要"打卡 30 天"（她已经被打卡 PUA 过了）

## 画像 2：在校大学生 / 应届生 小苏（占 30%）

### 基本盘
- **年龄**：18-23 岁
- **身份**：大三大四 / 研究生 / 应届
- **收入**：每月 1500-4000（生活费或第一份工资）
- **居住**：宿舍 / 合租，常去食堂
- **平台**：小红书 + 抖音 + B 站；公众号几乎不看

### 一天
- 9:00 起床赶课，吃食堂或随便买
- 12:30 食堂米饭 + 两菜
- 14:00 续命奶茶
- 18:30 食堂或外卖
- 22:00 宿舍夜宵（泡面、零食、果切）
- 凌晨 1-2 点睡

### 痛点
1. **学生党想瘦但没钱**：买不起轻食、办不起健身卡
2. **食堂选择有限**：天天油盐糖
3. **熬夜常态**：DDL + 室友作息 + 报复性夜生活
4. **第一次注意"皮肤 / 头发 / 经期"**：开始焦虑但缺知识
5. **被同龄人体型对比影响**：宿舍 6 个人 5 种身材
6. **PUA 减肥法首次接触者**：容易被"21 天减 10 斤"忽悠

### 我的稿子要给她的
- **省钱版**减脂法：食堂这么打、宿舍能囤的零食、三块钱能凑的"加餐"
- **不极端**的科普：保护她不进食障碍
- **学习/熬夜场景**专题：上自习吃什么、护肝护肤、月经不调原因
- **通用知识点**：先认识食物、再学搭配（她这阶段最需要"扫盲"）

### 标题模板
- 「学生党 30 块吃一周高蛋白早餐：宿舍能用的版本」
- 「食堂打饭 3 个原则：营养系学姐的避坑清单」
- 「宿舍夜宵实测：哪些不发胖还顶饿？」
- 「大学四年别再被这 5 个减肥谣言骗了」

### 不要对她做的事
- 不要让她买昂贵补剂、代餐、健身环
- 不要让她"早睡早起"（宿舍现实做不到）
- 不要复杂菜谱（她没厨房）

## 画像 3：宝妈 / 备孕期女性 阿楠（占 15%）

### 基本盘
- **年龄**：28-35 岁（边界放宽到 35）
- **身份**：备孕 / 孕期 / 产后 1-3 年
- **居住**：和老公或父母同住，有娃或准备有
- **平台**：小红书 + 公众号双平台，会主动搜索

### 一天
- 围着娃和家务转，吃饭时间不固定
- 老人做饭"重油重盐重淀粉"难统一
- 自己想恢复身材但被告知"哺乳不能减肥"

### 痛点
1. **产后体重 / 体型 / 体能**回不去
2. **母乳期能不能减、能吃什么**信息混乱
3. **婴幼儿辅食和家庭营养**全押在她身上
4. **甲状腺 / 缺铁 / 钙流失** 等产后高发问题
5. **婆媳/夫妻饮食观冲突**（老人的"养身菜" vs 营养学的搭配）

### 我的稿子要给她的
- 哺乳期 / 产后恢复期 **安全**的饮食与运动节奏
- 全家一起吃的"健康家常菜替换"方案（不是只为她一人备餐）
- 清晰的红线提示："这种情况必须去医院/找临床营养师"
- 关于补剂（DHA、铁剂、钙、维 D）的客观比较

### 标题模板
- 「产后 6 个月想减肥？哺乳期能做和不能做的事」
- 「家里老人爱做的 5 道菜，怎么改一下全家都更健康」
- 「备孕到产后，这 3 个营养素你必须盯紧」

### 不要对她做的事
- 不要在哺乳期话题下推任何减脂产品
- 不要谈"快速瘦回孕前"（极端不健康）
- 不要让她和老人对立（要给"温和沟通话术"）

## 共同的"不可触碰"红线

无论写给谁，都不允许：

- 鼓励 BMI < 18.5 的"瘦还要更瘦"
- 把任何食物贴"垃圾""毒"的标签
- 拿单一研究当全部真理
- 用"国外营养师 / 哈佛医生说" 当唯一依据而不给出处
- 让读者放弃就医
