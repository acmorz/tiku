## 三、写稿时 Claude 应该怎么用本库（操作流程）

### 步骤 1 — 选题阶段就识别"高频事实"
看选题，圈出可能涉及的具体数字 / 声明，例如：
- "每天该喝多少水" → 找 `中国居民膳食指南2022.pdf` 准则 6
- "经期能不能减肥" → 找 `中国居民膳食营养素参考摄入量2023.pdf` 铁部分 + `减肥科学原理`
- "奶茶有多少糖" → 找 `WHO-Healthy-Diet.pdf` 游离糖 + 膳食指南控糖
- "先吃菜还是先吃饭" → 找 `中国2型糖尿病防治指南2020.pdf` 进餐顺序

### 步骤 2 — 用 Read 工具直接读 PDF
Claude Code / Cursor 的 Read 工具原生支持 PDF。
不要"凭印象"回答数据，**所有具体数字必须开 PDF 看一眼再写**。

### 步骤 3 — 本库未覆盖时，按下列顺序兜底
1. 调用 `pubmed-search` Skill → 拿到 PMID
2. 国家级机构网站（CDC / WHO / NIH / 中国营养学会 / 国家卫健委）
3. 学会共识 / 教科书
4. **❌ 不接受**：自媒体二手转述、营销号、无作者博客、AI 凭印象编造的"研究表明"

### 步骤 4 — 引用格式
稿件中如使用本库内容，明确标注来源：

```markdown
> 来源：本地事实库 / 01-权威指南 / 中国居民膳食指南2022.pdf
> 原文出处：中国营养学会《中国居民膳食指南（2022）》
```

口播稿可以口语化但仍需出处：

> "中国营养学会 2022 版膳食指南建议，添加糖每天最好不超过 25 克，
>  也就是相当于半瓶可乐——你一杯全糖奶茶就已经超了。"

## 六、合规红线提醒

- 引用指南时不要简化为"标准 / 规定"，正确说法是"推荐"
- 不要把 RNI（推荐摄入量）和 UL（可耐受最高量）混淆
- 涉及糖尿病 / 高血压 / 孕期 / 哺乳期 / 进食障碍话题 → 必须加 "请咨询医生 / 临床营养师"
- 不能写"按指南吃就能停药"或"治愈某病"
